# commitToDo


# 概要

目標を明確にし、セルフイメージを高め、日々のタスクに取り組めるようにしたToDoアプリです

# 使い方

通常のToDoアプリとは違い、目標設定を終えないとタスクを登録できません。
目標設定とは、目標を明確化、理由の掘り下げ、セルフイメージを高める重要なプロセスです。

日々タスクに取り組む人をサポートするためにチャット機能も備わっています。
